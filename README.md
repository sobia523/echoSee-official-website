# EchoSee Official Website 👓✨

EchoSee is a futuristic, accessibility-first web platform for **EchoSee Smart Glasses** — an innovative AR solution designed to empower the hearing-impaired community by transforming speech into live, real-time subtitles projected directly onto AR lenses.


## 🚀 Key Features

- **Live AR Transcription Demo**: Experience the EchoSee interface with multilingual support (English, Urdu, Arabic, Chinese).
- **Interactive 3D Product View**: Explore the hardware with 360-degree rotation simulations.
- **Dynamic Pricing**: Toggle between individual and annual plans with real-time saving calculations.
- **Impact-Driven Storytelling**: Visualized global and local impact stats with count-up animations.
- **Premium Design**: Built with a "Digital AR" aesthetic using glassmorphism, high-contrast themes, and smooth Framer Motion animations.

## 🛠️ Tech Stack

- **Framework**: [Next.js 16 (App Router)](https://nextjs.org/)
- **Library**: [React 19](https://reactjs.org/)
- **Styling**: [Tailwind CSS 4](https://tailwindcss.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Icons**: [React Icons / Feather Icons](https://react-icons.github.io/react-icons/)

## 📦 Installation & Local Development

1. **Clone the repository**:
   ```bash
   git clone https://github.com/your-username/echosee-official-website.git
   cd echosee-official-website
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Run the development server**:
   ```bash
   npm run dev
   ```

4. **Build for production**:
   ```bash
   npm run build
   ```
